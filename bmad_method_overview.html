Chào mừng Agile AI.

BMAD (Breakthrough Method for Agile AI-Driven Development) là framework mã nguồn mở giúp biến AI từ một trợ lý chat lộn xộn thành một nhóm phát triển phần mềm có tổ chức, chuyên nghiệp và đáng tin cậy.

Bắt đầu ngay Đọc Tài liệu Docs

Tại sao bạn cần BMAD Method?

Hầu hết các lập trình viên đang dùng AI để "viết đại" mã nguồn (Vibe Coding) mà không có kế hoạch. BMAD mang lại cấu trúc và định hướng.

Cách cũ: "Vibe Coding"

  • Prompt: "Hãy tạo cho tôi một ứng dụng Todo..."
  • AI tự đoán mò yêu cầu và kiến trúc hệ thống.
  • Lỗi Context Window: Hôm sau mở lại, AI quên mất ngữ cảnh codebase.
  • Sửa chỗ này hỏng chỗ kia. Thiếu tài liệu nhất quán.

Cách mới: BMAD Method

  • Tạo tài liệu mô tả (PRD) và Kiến trúc rõ ràng TRƯỚC KHI code.
  • AI đóng vai trò như các chuyên gia (PM, Architect, Developer).
  • Lưu trữ Context: Mọi thứ được lưu thành tệp Markdown (.md) làm "bộ nhớ" vĩnh viễn cho AI.
  • Code theo từng chức năng nhỏ (User Stories), dễ test và dễ debug.

Các Trụ Cột Của Hệ Thống

Agent As Code

Các AI Agent (như Mary - Business Analyst, hay Murat - Test Architect) được định nghĩa hoàn toàn bằng các tệp Markdown (.md) độc lập, chứa cấu hình YAML. Dễ dàng chia sẻ và quản lý version.

Thích Ứng Mọi Quy Mô

Framework hỗ trợ 3 luồng công việc: "Quick Flow" cho việc fix bug nhỏ, "Standard" cho ứng dụng MVP, và các quy trình phức tạp cho hệ thống Enterprise.

Tích hợp IDE

Hoạt động trơn tru với các công cụ AI bạn đang dùng như Claude Code, Cursor, GitHub Copilot. Lệnh gọi `/bmad-help` giúp bạn tìm câu trả lời ngay trong IDE.

Quy Trình 4 Giai Đoạn (4-Phase Flow)

Cách BMAD chia nhỏ dự án lớn để AI không bị "ngợp".

1

Analysis
(Phân tích)

Agent Product Owner viết tài liệu yêu cầu sản phẩm (PRD), định nghĩa vấn đề và ràng buộc.

2

Planning
(Lập kế hoạch)

Agent Scrum Master chia nhỏ PRD thành các User Stories với tiêu chí nghiệm thu rõ ràng.

3

Solutioning
(Giải pháp)

Agent Architect quyết định công nghệ, cấu trúc thư mục và kiến trúc hệ thống tổng thể.

4

Implementation
(Triển khai)

Các Agent Developer & Reviewer viết code, tạo test, kiểm tra lỗi cho từng phần nhỏ.
